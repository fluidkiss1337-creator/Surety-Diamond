# Fluid-Innovation — Surety Compliance Diamond

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A modular, upgradeable on-chain compliance system built on the [EIP-2535 Diamond Standard](https://eips.ethereum.org/EIPS/eip-2535). Covers KYC, AML, sanctions screening, invoice registry, FATCA/CRS reporting, and multi-jurisdiction routing for enterprise supply chain finance platforms.

See [`fluid-compliance/README.md`](fluid-compliance/README.md) for full documentation, architecture, and deployment instructions.
